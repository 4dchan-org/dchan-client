dchan
-----

Decentralized Imageboard Client

! PoC !

## Development

### Requirements
- [`docker` and `docker-compose`](https://docs.docker.com/compose/install/)

### Start development
- `./bin/docker_bash.sh`
- In the container
    - `./bin/yarn_start.sh`
