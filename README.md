dchan
-----

Decentralized Imageboard Client

! PoC !

## Development

### Requirements
- [`docker` and `docker-compose`](https://docs.docker.com/compose/install/)

### Start development
- `docker-compose run --service-ports dapp bash` and `yarn && yarn start`
- `Open https://localhost:3000`
